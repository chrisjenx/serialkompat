# Security Policy

## Supported versions

serialkompat is in early development. Until a `1.0.0` release, security fixes are applied to the latest published version and `main` only.

## Reporting a vulnerability

Please **do not open a public issue** for security vulnerabilities.

Instead, report privately via [GitHub Security Advisories](https://github.com/chrisjenx/serialkompat/security/advisories/new). This lets us discuss and fix the issue before public disclosure.

We aim to acknowledge reports within a few days and to keep you informed of progress toward a fix. Thank you for helping keep serialkompat and its users safe.
